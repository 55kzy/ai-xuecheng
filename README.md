## 🎯 核心流程

```
方向选择(选1-2个)
    ↓
自适应摸底(3-6题, 6题型不重复, 题型混搭)
    ↓
客户画像生成(K/P/T/L/C 五维评分)
    ↓
专属课程定制(方向×水平×目标)
    ↓
EPIC教学(Explain→Practice→Insight→Consolidate)
    ↓
记忆追踪+遗忘提醒+效果迭代
```

## 🧠 题目格式

```json
{
  "id": "C1_G_K_01",
  "direction": ["C", "C1"],        // 大方向, 子方向
  "dimension": "K",                 // K工具认知/P提示词/T技术门槛/L信息素养/C概念理解
  "type": "G",                      // G梯度/S情景/P偏好/T两难/H习惯/C概念
  "baseLevel": 1,                   // 基准难度 0-3
  "text": "你让AI写文案，一般怎么描述要求？",
  "options": [
    {"text": "说个主题就行", "score": {"K": 0}},
    {"text": "主题+字数", "score": {"K": 1}},
    {"text": "主题+字数+语气", "score": {"K": 2}},
    {"text": "主题+字数+语气+参考风格+禁忌词", "score": {"K": 3}}
  ],
  "tag": ["文案", "prompt基础"]
}
```
