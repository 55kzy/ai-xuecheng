🎯 动手实操
15分钟 · 三步接到你第一个单
1 找到你的第一个渠道

别去淘宝、闲鱼竞争，那里太卷了。新手最适合的渠道是你身边的人和熟人圈子。

打开微信，做三件事：

  1. 发一条朋友圈："最近在学习用AI，可以免费帮忙做PPT/改文案/整理笔记，有需要的私我"
  2. 给3个关系好的朋友发私信："我正在练AI技能，你有啥需要AI帮忙的吗？免费帮你做一次"
  3. 如果有行业群/校友群/妈妈群，发一条："需要AI写文案/做PPT的吗？免费帮做，换点反馈就好"
为什么先免费？第一单的目的是跑通流程，不是赚大钱。免费反而能降低决策门槛，更快接到第一单。有了作品和流程，第二单就可以收费了。

如果你不想从熟人开始，也可以看看这些平台：

  • 🍠 小红书——发笔记"免费帮做xxx，限5个名额"，一般很快有人问
  • 🐟 闲鱼——挂"AI代写/代做PPT/代画图"，定价从最低的9.9元开始
  • 📱 微信群/知识星球——群里经常有人需要帮忙写东西做东西
2 接到需求后，让AI帮你干活

接到需求后，别慌。用这个框架跟客户确认：

要复述需求： "您是想要[具体内容]对吗？时间是[日期]，风格是[正式/轻松/专业]，字数大概[字数]，对吗？" 拿到确认后再开始做。避免做好后客户说"这不是我想要的"。

针对不同需求，用不同的prompt：

需求prompt模板
代写文案"你是一个专业的文案写手。我需要一篇[类型]文案，主题是[主题]，目标读者是[人群]。要求[具体要求]"
代做PPT"你是一个PPT内容策划师。帮我写一个[主题]的PPT大纲，共[X]页，每页列标题和要点"
代制图"请生成一张[风格]图片，内容是[描述]，比例[16:9/1:1/3:4]"
代翻译"你是一个专业翻译，把以下[语种]内容翻译成[目标语言]，保持原文语气和风格"
核心技巧：让AI先出初稿，你自己过一遍，改改明显不靠谱的地方，再发给客户。这不是作弊——你的人工审核和修改也是价值。
3 交付并为自己打样

交付时注意这几点：

  1. 准时——说好哪天就哪天，提前半天更好
  2. 超出一点预期——客户要500字方案，你给600字。多出来的这部分可能就是回头客的来源
  3. 要反馈——"您觉得哪里可以改进？下次我想做得更好"
  4. 留个口子——"后续还有需要随时找我"

做完第一单后，马上去做一个很重要的动作——把你这次做的东西整理成作品集：

把这次做的东西截图/存成PDF，放进一个叫"我的AI作品集"的文件夹。 下次接单时，发作品集给客户看，这就是你"有经验"的证据。
💡 理解为什么
5分钟 · "第一单"的真正价值不是那几十块钱

核心原理：从"我知道"到"我做过"

大多数人在AI搞钱上卡住，不是因为能力不够，而是因为完美主义——想着"等我准备好了再开始"。

等准备好的先开始的
心态"我还不够强""先做，边做边学"
进步研究了一个月，没行动做了一单，有了作品和经验
下一步继续研究可以涨价接第二单

第一单的价值不在于赚了多少钱，在于：

  • 你完整跑通了接单→做活→交付→收钱这个闭环
  • 你有了一个真实的作品案例
  • 你知道了自己的短板在哪（prompt不够好？沟通不够清楚？）
  • 你有了信心——"我能做到"

这个能力能迁移到哪

  • 涨价 — 有了第一单的作品，第二单可以涨价50%-100%
  • 批量 — 把相同的prompt存成模板，同类型需求快捷处理
  • 转介绍 — 老客户满意了，他会帮你介绍新客户
一个提醒：AI代做赚的是"信息差"的钱—会用AI的人帮不会用的人做。随着越来越多人学会用AI，这个差价会缩小。所以别只盯着代做，这是入门，不是终点。

5分钟 · 把第一步踩出去
今天做一次：选一个渠道（朋友圈/小红书/闲鱼），发一条免费代做的帖子。今晚睡觉前看有没有人联系你。没人联系？明天换个渠道再试一次。

进阶挑战

✅ 挑战1：整理你的服务清单
用AI帮你整理一个"我能做什么"的清单。打开AI说："列出10个用AI可以帮他普通人做的兼职服务，每项简述"。选3个你觉得最容易上手的做你的服务菜单。
✅ 挑战2：做个简易报价单
让AI帮你设计一个简单的报价模板："帮我设计一个AI代做的报价单模板，包括服务项目、价格区间、交付时间"。保存好，有人问就直接发。
✅ 挑战3：从1到3
第一单做完后，复盘整个流程，找出拖慢你效率的环节，用AI优化它（比如让AI写一个更全面的沟通话术模板）。

同类型迁移

"先做一单再优化"的思路还能用在：

  • 做小红书号 — 先发10篇笔记，再研究数据优化方向
  • 学新技能 — 先做出一个成品（哪怕很丑），再慢慢改进
  • 接长期客户 — 第一单做好了自然有回头客

⚡ 一劳永逸

存好这个流程，以后每次接单都这么走：

1. 发帖/发朋友圈 → "免费帮做XXX，限5个名额" 2. 收到需求 → 用确认话术跟客户对齐 3. AI干活 → 你过一遍 → 发给客户 4. 交付完 → 保存作品 → 问客户是否有长期需求
