2 注册与第一次对话

注册流程大同小异，大部分手机号就能搞定。以通用流程为例：

  1. 打开官网 — 浏览器搜索"豆包"（或想用的工具名），进入官网
  2. 注册/登录 — 用手机号注册，或者扫码登录（大部分支持微信直接扫）
  3. 进入对话框 — 注册完自动进入聊天界面，就跟微信聊天一样
  4. 写你的第一条prompt — 直接打字发过去

第一次对话推荐三条prompt（选一个试试）

💬 Prompt 1：认识AI "用大白话告诉我什么是AI？——假设我是一个完全不懂技术的高中生"
💬 Prompt 2：让它帮你规划 "帮我规划明天的三餐，预算60元，要健康，不用太复杂的菜"
💬 Prompt 3：日常润色 "把下面这段话改得更适合发朋友圈：[这里放你的话]"
重要心态：AI不是一次就能说准的。不满意？简单！就说"再改一版"或"换个风格试试"。AI不怕你反复改。
3 日常AI使用技巧包

学会这几个小技巧，你的AI使用效果直接翻倍：

🎭
改风格法：在prompt后面加"用[某某人/某某风格]的语气重写"——试试"用李佳琦的语气推荐早餐"
👤
角色扮演法："假装你是我的健身教练/营养师/财务顾问，帮我分析……"——AI会切换到专业人士状态
📋
示例驱动法：先给一个例子，再让它按同样的格式处理别的内容——"参考这个分析格式，帮我分析一下那个方案"
🔍
追问深挖法：每次回答后都追一句"为什么"——AI会一层一层解释，帮你彻底搞明白
📊
格式控制法："用表格/清单/时间轴呈现"——AI会自动调整输出格式，阅读体验翻倍
🔄
纠错提示法："上一版不对，原因是XX，换一个方向试试"——错了没关系，告诉它错在哪，它会调整
组合使用效果更好：比如"假装你是我的营养师（角色扮演），用表格（格式控制）帮我规划一周的健康食谱"。技巧叠加 = 效果翻倍。
💡 理解为什么
5分钟 · AI工具的使用本质：不是学App操作，是学一种沟通方式

好的AI用户 = 好的"任务描述者"

很多人刚用AI的时候，觉得"也就那样"——不是AI不行，是你没把话说明白。

你的表达越清晰、越具体、越结构化，AI的输出越好。同样的AI，给不同的人用，效果天差地别——区别就在沟通方式。

一个关键心法：把AI想象成"外国来的聪明实习生"

  • 它中文很好 — 能理解你说的话，语法也对
  • 但它不懂你的文化和语境 — 你说"随便搞一下"，它真的不知道"随便"是多随便
  • 它知识很多 — 读过无数资料，相当于一个行走的百科全书
  • 但它不会"猜你心里想什么" — 你不说清楚要什么格式、什么风格、什么用途，它就给你一个"差不多"的东西

同一个AI，不同用法

❌ 这样问（太模糊）
"帮我写个东西" → 结果：AI不知道你要写什么、给谁看、多长，给你一个四不像的段落
✅ 这样问（明确+具体+有格式）
"帮我写一条200字以内的朋友圈，主题是周末和朋友去爬山了，想要轻松幽默的风格，带一点对生活的感慨" → 结果：一次到位，基本不用改

看到了吗？不是AI变聪明了，是你把任务说清楚了。

记住这个公式：好的AI输出 = 明确的目标 + 具体的背景 + 想要的格式 + 不满意就追问，再多说一句"换个版本试试"。
🔁 巩固练习
5分钟 · 三个挑战，巩固你的实操能力
马上试：不用等明天，现在就找一台电脑或手机，把下面任意一个挑战做了。

进阶挑战

✅ 挑战1：注册一个新AI工具
选一个你没用过的工具（比如上面对比表格里你没试过的），注册后问它一个"需要分析"的问题，比如"帮我分析一下，上班通勤1小时vs在公司附近租房，各自的优缺点是什么"。感受不同工具的回答风格。
✅ 挑战2：组合技巧写一条高级prompt
把至少3个技巧组合成一条prompt。比如："假装你是我的营养师（角色扮演），用表格（格式控制）帮我规划一周的减脂食谱，参考下面这个例子（示例驱动）：[随便贴一个膳食搭配]"。看看组合后的效果是不是比单个技巧好用。
✅ 挑战3：连续追问5次"为什么"
选一个话题（比如"为什么飞机票越临近出发越贵"），让AI解释。然后每次回答后都追问"为什么"，连续问5次。观察AI的回答质量变化——是一次比一次深入，还是有重复或胡说的地方？这能帮你理解AI的上下文长度边界在哪里。

⚡ 记住一句话

**会跟AI沟通 = 会跟陌生人沟通：明确目标 + 说清背景 + 告诉它你想要的样子**

学会AI不是学会一个App操作，是学会一种沟通方式。 三步上手法： 1. 选工具 → 注册2~3个免费AI工具 2. 写prompt → 明确、具体、有格式 3. 迭代优化 → 不满意就说"再来一版" 最后记住：AI不是魔法，是一个"外国来的聪明实习生"。 你说得越清楚，它做得越好。
