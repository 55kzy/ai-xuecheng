📌 为什么你的脚本总是写不出来
3分钟 · 先搞清卡在哪一步

很多人想做短视频，打开文档对着空白的编辑框，大脑也一片空白。

不是你没内容可讲，是你把"写脚本"这件事想得太重了——总想一次性写出完美的台本，结果越想越写不出来。

换个思路：让AI帮你把框架搭好，你只需要往里面填你自己的观点和经验。一篇文章、一段录音、甚至几个关键词都能变成脚本。

❌ 大多数人这样跟AI说
"帮我写个短视频脚本"
✅ 给出素材和方向效果完全不同
"我想做一个讲AI干货的短视频，主题是AI怎么做PPT。给我写个45秒口播脚本，开头抓眼球、中间讲实用操作、结尾总结"

差的 prompt → 笼统的脚本 → 你觉得没用。
好的prompt → 你可以直接用的框架 → 填上你的内容就完事。

15分钟 · 三步把AI变成你的脚本助手
1 先给框架再给内容

不要指望AI一次就写出完美脚本。先让它给你一个结构模板，你再往里填自己的料。

你是一个短视频脚本写作助手。帮我生成一个[口播类型]视频的脚本框架。 视频主题：[写你的主题，比如"用AI做PPT"] 目标观众：[谁在看，比如"职场人"] 时长：[30秒 / 45秒 / 60秒] 风格：[干货型 / 故事型 / 教程型] 框架包含：开场钩子→核心内容（分点）→结尾号召。每段标注预计时长。

AI给你框架后，你再告诉它每个部分你的具体内容：

现在帮我填内容。开场钩子就用这句："你是不是每次做PPT都要熬到半夜？" 核心内容写三点： 1. 打开AI工具，把你要的内容描述给它 2. AI生成后，让它帮你浓缩成一页三行 3. 复制进去，调字体和颜色就完事 结尾号召：关注我，下期教你用AI做汇报PPT

你会发现，这样出来的脚本不是AI的"创作"，而是你的观点 + AI的结构。

2 把一篇文章变成脚本

如果你写过公众号、知乎、小红书，上面的内容可以直接「翻译」成视频脚本，不用重写。

把你已有的文字稿贴给AI：

以下是一篇关于"怎么写prompt"的文章，请把它改写成45秒的短视频口播脚本。 要求：保留核心观点，用口语化表达，每句话不超过15个字。 开头用一个反问句抓人。 [粘贴你的文章]

这个方法的好处：你已经有内容了，只是换一种形式。AI帮你做"格式转换"，你省了从零构思的时间。

3 做你自己的脚本模板

最后这步很关键——把你常用的脚本类型做成模板，以后每次填关键词就出。

存这个到备忘录：

你是一个[类型]短视频脚本助手。帮我写[时长]脚本。 主题：[主题] 目标观众：[人群] 风格：[风格] 结构要求： 1. 开场钩子（5-7秒，用反问/数据/痛点） 2. 核心内容（分3点，每点加一个小标题，总长25-35秒） 3. 收尾总结+行动号召（5秒） 输出格式：每句一行，标注建议画面或表情。 参考风格：[可选，给一个你喜欢的UP主或风格说明]

存到备忘录/飞书/Notion里，每次想做哪个方向的视频，填空就行。

小技巧：如果你有自己拍的爆款视频，把它的文案喂给AI，让它提炼模板，以后再按这个风格生成新的脚本。
💡 理解为什么
5分钟 · 你不是在学"写脚本"，是在学"给AI搭框架"

这节课的核心原理就三个字：先搭架子

很多人让AI写脚本效果不好，是因为把该自己做的事丢给了AI。AI不懂你的个人风格、不知道你积累了什么案例、不清楚你的观众是谁。

AI的局限。好的AI协作不是"AI帮我写"，而是"给我结构我来填"或"我出内容AI帮我组"。

差的用法好的用法
方式"帮我写脚本""这是框架，帮我填内容" 或 "这是我写的内容，帮我编成脚本"
结果AI的脚本，你不想用你的内容，AI的结构，你愿意录
时长改到不想发3分钟出一版，微调就开录

这个套路能迁移到哪

  • 做教程视频 — 把你会的步骤写成bullet points，让AI帮你扩展成完整的讲解词
  • 测评/开箱 — 列出你发现的产品优缺点，让AI帮你排好逻辑顺序
  • 直播话术 — 给产品卖点和常见问题，让AI帮你写成开场+讲品+逼单三段
  • 口播练习 — 让AI把脚本改成"说人话"版本，方便你对着念
一个提醒：AI写出来的脚本可能太"像AI"——太规整、太完美、缺少真实的口语气感。拿到脚本后，用自己的话顺一遍，改成你平时说话的节奏。观众要的是"像人在说话"，不是"AI写的完美稿"。
🔁 巩固练习
5分钟 · 连做三次变成习惯
今天睡前做一次：打开AI工具，用三步法（框架→填内容→出脚本）生成一个60秒口播脚本。主题就用你最近想做的那个。不要看教程，卡住再翻回来。

进阶挑战

✅ 挑战1：多版本输出
同一个主题，让AI给你写三个不同风格的版本（干货型/故事型/搞笑型），看看同一个内容不同讲法的差别。
✅ 挑战2：从评论挖选题
复制你同领域爆款视频底下的评论发给AI，让它从评论里提炼新视频选题。需求往往藏在评论区。
✅ 挑战3：先录音再转脚本
不写脚本直接打开录音，把想说的内容说一遍。把语音转文字后扔给AI："帮我整理成结构化的口播脚本"。

同类型迁移

"给AI搭框架"的思路还能用在：

  • 写直播话术 → 给产品信息和常见Q&A → "帮我写成开场+介绍+逼单的直播话术"
  • 做产品介绍页 → 给产品三个卖点 → "按AIDA模型写一段产品介绍"
  • 写课程大纲 → 给你会的知识点 → "帮我排成一门课的大纲，每节15分钟"

⚡ 一劳永逸

存好上面的 prompt模板，以后每次拍视频流程：

1. 打开备忘录 → 选要用的脚本模板（口播/教程/测评） 2. 填上主题和你的核心内容 → 发给AI 3. 拿到脚本 → 用自己的话顺一遍 → 开录
🎉 以后你的脚本，从想到出不超过10分钟
