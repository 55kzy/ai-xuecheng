3分钟 · AI早就不是"聊天的"了

很多人对AI的认知还停留在"就是那个能聊天的"。但现在的AI世界已经远远不止于此。

AI可以帮你：

  • 写东西 — 文案、邮件、报告、朋友圈
  • 做图 — 一句话生成插图、海报、logo
  • 做视频 — AI生成短视频、配音、字幕
  • 办公 — 做PPT、分析数据、整理表格
  • 学习 — AI老师一对一辅导、翻译、总结
  • 编程 — 不懂代码也能做个小工具

这节课不是教你怎么用某款工具，而是给你一张"AI工具版图"——让你知道AI现在能做到什么程度、有什么工具可以用。

15分钟 · 10个AI工具，每个场景至少有一个
1 写作 + 学习类AI

1. 豆包 / Kimi / 通义千问

国产三巨头，免费、中文好、有App。适合日常对话、写作、总结、翻译。

用法：手机应用商店搜名字下载 → 注册 → 直接聊。

2. ChatGPT

综合最强的聊天AI，能写能聊能推理能画图。部分免费，完整能力需付费订阅。需要特殊网络访问。

3. DeepL / 沉浸式翻译

翻译类AI。DeepL翻译质量最高，沉浸式翻译是浏览器插件，看外文网页时一键对照翻译。

该选哪个？只用一个 → 豆包（免费+中文好）→ 需要特殊访问能力 → ChatGPT。翻译用DeepL+沉浸式翻译插件。
2 图片 + 视频 + 办公类AI

4. Midjourney / DALL·E / 通义万相

AI绘画三强。MJ效果最好（需付费），DALL·E内嵌ChatGPT（方便），通义万相（免费中文）。一句话生成图片/插图/海报。

5. 剪映 / Sora / 可灵

AI视频工具。剪映有大量AI功能（文字转语音、自动字幕、智能抠像）。可灵（快手）能做AI视频生成。

6. Gamma / 美图AI PPT

AI做PPT。一句话生成精美PPT，适合不想花时间排版的人。

7. Notion AI

AI笔记+文档+数据库。一边写笔记一边让AI帮你改写、翻译、总结。

先试哪个？做图 → 通义万相（免费）。做视频 → 剪映（你手机上可能已经有了）。做PPT → Gamma 免费版试试。
3 编程 + 实用工具类AI

8. Claude / 文心一言（插件生态）

编程AI。Claude写代码能力强，但需要特殊网络。国内可以做AI编程的还有CodeGeeX等。

9. Perplexity / 天工AI搜索

AI搜索引擎。不是给你一堆链接，而是直接回答你并附来源。做研究查资料的好帮手。

10. Grammarly / 写作猫

AI写作助手。Grammarly改英文语法（有浏览器插件），写作猫是中文AI写作辅助。

5分钟 · 工具很多，但核心规律就几条

AI工具的核心规律

  1. 聊天类AI最好先上手 — 豆包/Kimi/ChatGPT，先会一个再学其他的很容易
  2. 专一工具比通用工具在某方面更强 — 做图选Midjourney vs 用ChatGPT做图，MJ效果碾压
  3. 免费的往往够用 — 大多数人的需求国产免费工具都能满足
  4. 插件让工具更好用 — 沉浸式翻译、Grammarly这种插件装一次，所有网页都能用
  5. 工具更新很快 — 半年前的教程可能已经过时，遇到具体需求直接搜"XX AI工具 2026"
建议顺序：先用好1个聊天AI → 加1个图生图AI → 再加1个浏览器插件 → 遇到具体需求再找专精工具。不要一开始同时学10个。

进阶挑战

✅ 挑战1：同一个任务，不同工具
让Kimi和豆包同时帮你做同一件事（比如写一篇文章），对比输出质量。你会开始理解每个工具的"性格"。
✅ 挑战2：AI做PPT实操
用Gamma或美图AI PPT，输入一个你需要的主题，看看AI生成PPT的水平。你会惊讶于它的完成度。
✅ 挑战3：搜索对比
同一个问题，分别用天工AI搜索和Google/百度搜，对比答案的差异。你会发现AI搜索在"找答案"而不是"找链接"上的优势。
