15分钟 · 三步把AI从"随便说说的"变成"懂你的"
1 说清三要素：谁 + 场景 + 偏好

日常场景让AI给建议，你只需要多说三个东西：

  1. 对象 — 给谁用的？你自己？带爸妈？跟朋友？有小孩吗？
  2. 场景 — 什么场合？约会/家庭聚会/一个人发呆？
  3. 偏好 — 你的口味/风格/预算/时间限制
# 差 推荐个番剧 # 好 最近剧荒，求推荐。我喜欢悬疑烧脑的，单元剧最好，最近看了《胆小鬼》觉得节奏太慢。别推荐《开端》这种看过的主流热门。

看到区别了吗？好问法给了AI三条线索——类型偏好、看过的不喜欢的、排雷热门款。

试试看：打开你的AI，把上面"好"的问法发过去，看看它给你的推荐是不是比平时准很多。
2 不满意？别重问，给反馈

日常用AI最容易犯的错：AI给了一个不行的答案 → 关掉重开 → 又问一遍差不多的 → 又得到一个不行的答案。

别关！直接在同一个对话里追加反馈，AI会记住你的偏好并调整：

# AI推荐了A/B/C三部 看过A了，B的画风不喜欢，C可以进一步说说吗？ # AI推荐了川菜馆 有没有不用排队的？上次等了1小时。 # AI帮写的文案 太正式了，我朋友群发不需要这么严肃，活泼一点

追加反馈比重头来过快10倍，而且AI会越聊越懂你——因为同一个对话上下文里，你的偏好它全记得。

小技巧：顺便加一句"以后都按这个风格"，下次同话题直接继承。
3 几个高频场景的万能问法

日常场景不用每次都从头想prompt，记住几个套话就行：

🍽 吃啥

我在[地点]，[人数]个人，[场合]，人均[预算]， 想吃[菜系]，帮我推荐，要求：[任何额外条件如停车/包厢/不辣]

🌍 去哪玩

我从[出发地]去[目的地]，[天数]天，带[谁]， 喜欢[自然/人文/美食]，不要[什么类型的活动]， 帮我排个不赶路的行程

🎬 看什么

最近想看[类型]，喜欢[参考作品A]和[B]，讨厌[C]，入坑指南也给我一份

✍️ 发朋友圈

我去了[地点]，拍了[描述照片]，感受是[一句话心情]， 帮我写几条朋友圈文案，风格[文艺/幽默/凡尔赛/极简]， 每条不超过50字

把这些存到备忘录里，以后吃饭/旅游/发圈直接填空就行。

💡 理解为什么
5分钟 · 日常场景prompt的核心原理就四个字

信息越多，AI越准

AI不是读心术，它没有你的记忆。你跟说"推荐个餐厅"在它看来跟另一个用户说"推荐个餐厅"没区别。

但只要你说"预算150、川菜、别太辣、有包厢"，它的推荐池就被缩到了原来的1%甚至更少——精度自然上去了。

模糊问法清晰问法
问法"推荐个电影""喜欢悬疑，看过《消失的她》觉得太套路，求推冷门但高分的"
结果豆瓣Top250，你八成看过可能会有你没看过的宝藏
效率花10分钟翻+3次重问1次到位

这个套路能迁移到哪

  • 买东西决策 — "我要买XX，预算XX，看重XX，跟YY比哪个好"
  • 写小作文 — "写一段道歉的话/面试自我介绍/相亲简介/感谢信"
  • 日常学习 — "我完全不懂XX，用小孩能听懂的语言解释，举个生活中的例子"
  • 纠结时 — "A和B各有啥优劣，帮我从[角度]分析选哪个"
注意：日常场景AI也可能出错。它说"这家店评分4.8"，你最好自己搜一下确认。建议/推荐类可以信，事实/数据类要核实。

5分钟 · 明天用起来才真的学会
明天做一次：下次你纠结吃什么/看什么/买什么的时候，打开AI，用三要素法（对象+场景+偏好）问一次。别翻教程，卡了再回来看。

进阶挑战

✅ 挑战1：反向过滤
让AI先推荐5个，然后说"去掉我不喜欢的XX类型/风格，再推荐3个"——体验连续对话的力量。
✅ 挑战2：让AI帮你砍价&比价
发一个商品链接或描述给AI，说"帮我在XX平台找一个更低价的同款"，或"帮我写一段砍价话术"。
✅ 挑战3：一份完整的周末计划
把你下周末的真实空闲时间、预算、想干什么（哪怕只是"想躺着")发给AI，让它出一份计划。不满意就追加上一条的反馈技巧。

⚡ 记住一句话

**日常场景 = 对象 + 场景 + 偏好 + 排雷**

1. 遇到纠结的事 → 打开AI → 说清三要素 2. 不满意 → 追加反馈 → 别重开 3. 把高频问法存备忘录 → 填空用
🎉 以后AI给你的推荐，不会再让你"不感兴趣"
